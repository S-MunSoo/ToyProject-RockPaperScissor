### React 첫 토이 프로젝트 가위바위보 게임

#### 구현 로드맵

- 1. 유저는 박스 두개를 볼 수 있다.
- 2. 박스에는 타이틀이 있고, 사진정보 , 결과 값이 들어간다
- 3. 가위바위보 버튼이 보인다.
- 4. 버튼을 누르면 클릭한 아이템이 유저눈에 보인다
- 5. 컴퓨너틑 랜덤하게 선택이 된다.
- 6. 3, 4번의 결과를 가지고 누가 이겼는지 승패 결정한다
- 7. 승패에 따라 박스테두리 색이 바뀐다(이기면:초록 , 지면:빨강 , 무승부:블랙)

#### 사용기술스택

- HTML
- CSS
- JavaScript
- React
